# 📘 Assignment: Exploratory Data Analysis with Pandas

## 🎯 Objective

Use Python and Pandas to explore a dataset, clean the data, compute summary statistics, and create visualizations that reveal insights.

## 📝 Tasks

### 🛠️ Load and Inspect Data

#### Description
Load a dataset into a Pandas DataFrame and inspect its structure and contents.

#### Requirements
Completed program should:

- Read a CSV dataset into a Pandas DataFrame.
- Display the first few rows of the dataset.
- Show the dataset shape, column names, and data types.
- Count and report missing values for each column.

### 🛠️ Clean and Prepare Data

#### Description
Clean the dataset by handling missing values and converting columns to appropriate data types.

#### Requirements
Completed program should:

- Fill missing values or remove rows/columns with missing data.
- Convert one or more columns to the correct data type where needed.
- Create a new column derived from existing data.
- Output the cleaned DataFrame summary.

### 🛠️ Analyze and Visualize Insights

#### Description
Compute summary statistics and build simple visualizations to highlight important trends.

#### Requirements
Completed program should:

- Calculate summary statistics like mean, median, and counts for key columns.
- Group data by a categorical column and compare results.
- Create at least one chart (bar, line, or histogram) from the cleaned data.
- Explain one insight discovered from the analysis in a short summary.
